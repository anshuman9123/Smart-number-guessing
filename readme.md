# 🎯 Number Guessing Game (Python)

An interactive command-line based number guessing game built using Python.
This project demonstrates logic building, user interaction, and the use of the Binary Search algorithm for efficient guessing.

---

## 🚀 Features

* 👤 **User Guess Mode**
  User tries to guess a randomly generated number (1–100)

* 🤖 **Computer Guess Mode**
  Computer intelligently guesses the number using **Binary Search**

* 🔁 Real-time feedback system (`higher`, `lower`, `correct`)

* 📊 Tracks number of attempts taken to guess correctly

---

## 🛠️ Tech Stack

* Python (Core Logic)
* Random Module
* CLI (Command Line Interface)

---

## 🧠 Concepts Used

* Conditional Statements
* Loops (while loop)
* Functions
* Binary Search Algorithm (O(log n))

---

## ▶️ How to Run

```bash
python main.py
```
